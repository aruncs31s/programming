## MATLAB


`clear all;` -> Used to clear the workspace

`close all;` -> used to close all the plots and graphs

`clc;`  -> clear the command line


### plot sine wave
```
t = -2*pi:pi/100:2*pi

plot(sint(t))
```

### plot cosine wave

